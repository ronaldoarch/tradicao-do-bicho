# Tradição do Bicho

Site de apostas do Jogo do Bicho - Réplica do site pontodobicho.com

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Swiper.js

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador

## Estrutura do Projeto

- `/app` - Páginas e layouts do Next.js
- `/components` - Componentes React reutilizáveis
- `/public` - Arquivos estáticos (imagens, etc)

## Componentes Principais

- `Header` - Cabeçalho com logo e navegação
- `HeroBanner` - Banner promocional principal
- `StoriesSection` - Seção de stories
- `LiveQuotation` - Cotação ao vivo com carrossel
- `JackpotSection` - Seção de jackpot com contador
- `VipBanner` - Banner VIP
- `FAQSection` - Seção de dúvidas frequentes
- `Footer` - Rodapé
- `BottomNav` - Barra de navegação inferior (mobile)
